    学习目标与事实状态

    学完本章后，应能从 J1 的引脚连接解释 USB Type-C 可翻转、CC 检测和 VBUS 供电，并能说明 R1、R2 与 C1 的作用、布局重点及失效后果。

    • 【设计】表示最终 R1 工程中采用的网络、标称值或装配状态。
    • 【计算】表示由设计值和公式推导的结果。
    • 【预期】表示依据电路机理推断、尚未由实物确认的行为。
    • 【待验证】表示仍需数据手册细节、真实主机/线缆或装配板实验确认。

    当前事实库中的实测记录为 0；本章不会把标称 5 V、计算电流或预计现象写成实测结果。

    7.1 USB-C 连接器对 USB 2.0 设备意味着什么

    R1 的 J1 是 TYPE-C-31-M-12、16 引脚、仅 USB 2.0 的 Type-C 母座【设计】。它同时承担三类任务：把主机的 VBUS 引入 VBUS_IN，用 CC1/CC2 表明本板是受电设备，并把 USB 2.0 的 D+、D− 接入数据通道。它没有 SuperSpeed 通道，也没有 USB Power Delivery（USB-PD）控制器。

    J1 在最终 R1 工程中的关键连接【设计】
    J1 引脚组网络用途
    A4/B9、B4/A9VBUS_IN主机提供的总线电源输入。
    A1/B12、B1/A12GND供电回流和信号参考。
    A5CC1插头方向、角色和供电能力检测的一侧配置通道。
    B5CC2另一插头方向对应的配置通道。
    A6、B6USB_DP_CONN可翻转后的 USB D+。
    A7、B7USB_DM_CONN可翻转后的 USB D−。
    A8、B8未连接R1 不使用这些引脚。
    外壳固定焊盘 1–4GND机械固定，并在 R1 中直接接公共地。

    J1 位于左侧板边、朝外并有意略微伸出板框【设计/计算】。连接器方向、外壳固定脚和插拔空间属于机械约束；即使电气网络全对，方向装反或焊盘错位仍会使成品无法插接。

    安全边界：本板与 USB 主机共地且没有隔离。不要把 J1 的地、目标板的地和其他已接地仪器随意组成未知回路；接线前先确认各设备的参考地关系。

    7.2 可翻转的 D+ 与 D− 引脚

    Type-C 插头可以正插或反插，因此母座为 USB 2.0 数据各提供两只接触点。R1 在连接器处把 A6 与 B6并到 USB_DP_CONN，把 A7 与 B7 并到 USB_DM_CONN【设计】。无论插头方向如何，后级 U2、R3/R4 和 U1 看到的仍是一对 D+/D− 网络。

    “并在一起”只解决方向复用，不代表可以任意拉长支路。连接器焊盘到合并点、ESD 器件和后续差分走线仍应保持短、对称并具有连续地参考；第 8 章将分析完整数据路径。

    常见误解：CC1/CC2 不能像 D+ 的 A6/B6 那样直接短接。两根 CC 线分别代表两种插头方向，必须各自保留独立检测支路。

    7.3 CC1 与 CC2 如何完成连接检测

    配置通道（Configuration Channel，CC）先于 USB 数据枚举工作。作为受电端，R1 在 CC1 和 CC2 上分别放置受电端下拉 Rd。供电端在自己的 CC 上提供上拉；插入后，供电端从其中一根 CC 检测到 Rd，从而知道：

    1. 有一个受电设备接入；
    2. 哪一根 CC 是当前有效通道，也就是插头方向；
    3. 应按 Type-C/USB 规则提供 VBUS，并通过 CC 表达可用供电能力。

    另一根 CC 在该插头方向下不承担方向检测，但它必须仍有自己的 Rd，才能保证翻转插头后同样被识别。CC 检测与 USB 设备枚举是两个阶段：前者建立连接角色和供电条件，后者由 U1 的 USB 协议栈与主机交换描述信息。

    7.4 R1 与 R2：为什么需要两只独立 5.1 kΩ 下拉

    CC 下拉元件事实
    位号值与网络作用与布局失效影响
    R1 5.1 kΩ，0603；CC1—GND【设计】 CC1 的独立 Rd；靠近 J1，避免长支路和误接【设计】。 开路或错大：一个插头方向可能无法识别或供电；短路或错小：CC 电流异常【预期】。
    R2 5.1 kΩ，0603；CC2—GND【设计】 CC2 的独立 Rd；必须与 CC1 电气分离【设计】。 开路或错大：另一个插头方向可能失败；短路或错小：CC 检测异常【预期】。

    两只 5.1 kΩ 电阻不是并联成 2.55 kΩ，因为 CC1 与 CC2 是两个独立网络。把它们短接会破坏方向检测；只装一只则会造成明显的单方向故障。

    7.5 VBUS、连接器外壳与地引脚

    J1 的 VBUS 焊盘汇入 VBUS_IN【设计】，再分别供给 U3、U4 及其局部电容。所有电流都必须经 GND 返回主机；地不是“吸收电流的点”，而是完整回路的一半。

    C1 是 10 µF、C0805 的 VBUS 本地储能/旁路电容，连接 VBUS_IN—GND【设计】。它与 C2、C13 在原理图上并联，但物理位置不同，分别服务连接器附近、Buck 输入和 U4 输入的局部电流环。C1 开路时，VBUS 瞬态和噪声裕量预计下降；C1 短路时，主机 VBUS 被对地短接，主机端口保护可能动作【预期，待实物验证】。

    最终 R1 把 J1 的四个外壳固定焊盘直接接 GND【设计】；早期提案中的可配置 0 Ω/RC 外壳网络没有出现在最终 48 个位号中。直接接地能形成明确的机械与电气参考，但真实 ESD、EMI 和机壳电流路径仍为【待验证】。

    7.6 枚举前后都不能忽略 USB 电流预算

    连接建立后有三个不同概念：

    1. CC 检测：确认受电角色、方向和供电声明；
    2. VBUS 上电：给 U3、U1 和其他板上电路供电；
    3. USB 枚举：U1 作为 USB Full-Speed 设备向主机报告身份和功能。

    R1 的 5V_SW 目标电流为 0.4 A，3V3_SW 目标电流为 0.25 A【设计】，但这些目标不能覆盖主机端口、线缆、电源转换、板上消耗和枚举策略的约束。实际可用电流、线缆压降、主机保护动作和热性能均为【待验证】。安全固件应在复位、启动和 USB 挂起时保持两路开关关闭【预期，硬件/固件尚未验证】。

    不得这样做：不要因为看到 VBUS 标称 5 V，就把 5V_SW 与 3V3_SW 的目标电流简单相加并认定任意 USB 口都能持续供电；也不要在枚举和安全状态尚未验证时接入高功率目标负载。

    7.7 为什么 R1 不实现 USB Power Delivery

    USB-PD 需要在 CC 上进行专门协议通信，并要求相应控制器、功率路径和协商状态机。R1 只有 J1、R1/R2 的受电端 Rd、USB 2.0 数据通道和普通 VBUS 电源路径【设计】；没有 PD 控制器，也没有可协商高压电源轨。

    因此，本板应被理解为USB 总线供电的 USB 2.0 设备，而不是可请求更高电压或任意功率的 PD 受电器。把 PD 充电器接上并不会自动把 R1 变成 PD 设备；没有协商时不得假定出现高于普通 VBUS 的电压。

    7.8 连接器外壳接地的工程取舍

    外壳接地要同时考虑机械强度、静电泄放、共模电流、机壳结构和系统接地。R1 是无金属机壳、无隔离的两层小板，最终选择外壳焊盘直接接公共 GND【设计】。这一连接在 CAD 中明确，但它不能单独证明 ESD 或 EMI 合规。

    布局审核应检查：

    • 外壳固定焊盘有足够铜和可靠焊接，且不误连 VBUS；
    • 静电回流尽量在连接器入口附近闭合，不穿过 U1、Y1 或 BUCK_FB 的敏感区域；
    • 连接器外伸、线缆插拔和外壳受力不会撬动小信号焊盘；
    • 示波器、电脑和目标板同时接地时没有危险的地环路。

    7.9 实验：识别 USB-C 引脚并测量 CC 电阻

    目的与器材

    在不上电条件下确认 J1 的可翻转数据引脚、CC1/CC2、VBUS 和 GND，并验证 R1/R2 的直流下拉路径。器材：R1 原理图、带 Type-C 引脚标注的无源转接板或测试夹具、数字万用表和防静电工作台。

    安全与停止条件

    • 拔掉 USB 线、目标板、调试器和所有外部电源；电阻档绝不能接到带电电路。
    • 不要用粗探针直接撬压 J1 的细间距触点；优先使用无源转接板。
    • 若 VBUS_IN—GND 接近短路、CC 对地接近 0 Ω，或读数不断异常漂移，停止并检查焊桥、错件和测量路径。

    步骤

    1. 用导通档确认 J1 的地引脚和外壳焊盘属于 GND；不要把蜂鸣声当作低阻精确值。
    2. 确认 A6/B6 同属 USB_DP_CONN，A7/B7 同属 USB_DM_CONN；确认 D+ 与 D− 之间不短路。
    3. 改用合适的电阻档，分别测 CC1—GND 和 CC2—GND。
    4. 记录读数、量程、探具和板号；再交换表笔重复一次，以发现外部并联路径或接触问题。

    本章小结

    • J1 用重复的 D+/D− 触点实现 USB 2.0 可翻转，但 CC1/CC2 必须保持独立。
    • R1、R2 各为 5.1 kΩ Rd，缺一只会形成方向相关故障；C1 为 VBUS 的 10 µF 本地储能。
    • CC 检测、VBUS 上电和 USB 枚举是不同阶段；R1 没有 USB-PD。
    • 最终外壳直接接 GND；真实供电能力、ESD、EMI 和枚举行为仍需实物验证。
