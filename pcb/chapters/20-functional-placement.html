    20. 功能化布局（Functional Placement）

    学习目标

    功能布局把原理图关系变成电流回路、信号路径、热路径和可装配空间。完成本章后，你应能按照“机械固定对象 → 入口保护 → 关键回路 → 重复通道”的顺序布局，并在锁定位置前完成机械审核和受影响布线清理。

    • 先固定连接器和机械约束，再放电气关键元件；
    • 缩小 USB 未保护段、Buck 高 di/dt 回路和去耦回路；
    • 让晶振、负载开关和重复 GPIO 通道具有清晰的功能邻接；
    • 用统一间距组织无源器件，而不是只追求最小面积；
    • 从 J1、R1、C8/C9 和陈旧走线案例建立变更纪律。

    20.1 先放连接器与机械约束

    J1、J2、J3 决定线缆从哪里进入、用户从哪里操作，也决定许多信号与电源的起点。它们应在高密度器件之前放置并约束。

    1. J1 固定在左边缘，开口朝左、旋转 −90°，只保留有意壳体外伸；
    2. J2 固定在右边缘，旋转和 1 脚方向与目标线束一致；
    3. J3 放在上边缘附近，给调试插头或探针留出垂直空间；
    4. 标出 USB 插头、目标线束、探针、手指和未来外壳的禁止区；
    5. 锁定经过机械审核的连接器，后续移动必须触发重新审核。

    R1 最终 J1 中心为 (200.5, 472) mil，J2 约为 (1850, 472) mil，J3 位于上方调试区。坐标不是布局原则本身；关键是方向、板边和外部配合。

    20.2 把 ESD 保护放在入口处

    U2 的职责是在外部静电事件进入长板内走线之前提供低电感泄放路径。因此放置顺序是 J1 → U2 → R3/R4 → U1，而不是先围绕 MCU 填满器件后再找空位。

    • 缩短 J1 到 U2 的 USB_DP_CONN/USB_DM_CONN 未保护段；
    • 让 D+、D− 到 U2 的拓扑对称，避免支路和测试焊盘形成桩线；
    • 让 U2 的 GND 回路短、宽，并就近连接连续地参考；
    • 不要让 ESD 电流先经过 MCU 区域或细长地线再到地；
    • 同时给 J1 外壳固定脚和 USB 扇出留出制造间距。

    最终 U2 位于 (545, 350) mil。这个坐标是组织化布局的结果，不代表只要复制坐标就自动得到同样的回流和阻抗。

    20.3 把 Buck 变换器放成紧凑电流环

    Buck 布局至少有四条不同性质的路径：输入脉冲环、BUCK_SW 开关节点、L1—C3 输出环和高阻抗反馈路径。把它们画成原理图上的一个方框不够，必须在 PCB 上分开优化。

    1. C2 紧靠 U3 的 VIN/GND，缩小输入脉冲电流环；
    2. L1 紧靠 U3 SW，引脚间 BUCK_SW 铜短且面积小；
    3. C3 靠近 L1 输出和 U3 地返回，形成短输出环；
    4. R5/R6 与 FB 引脚靠近，反馈线从安静输出点取样；
    5. 让 BUCK_SW 远离 USB、晶振、NRST 和反馈；
    6. 预留输入/输出主干宽度和 U3/L1 的散热铜，而不把开关节点无限铺大。

    布局紧凑不是把所有器件任意挤在一起。若 C2/C3 的焊盘或本体与 U3/L1 冲突，实际装配和电流路径都会变差；机械间距与电气回路必须同时满足。

    20.4 把去耦电容放到其服务引脚旁

    去耦的评价对象是“电源脚 → 电容 → 地 → 芯片地脚”的高频回路，而不是电容中心到 MCU 中心的直线距离。每个电源脚优先拥有自己的近端 100 nF 路径，再由 C9 承担较低频储能。

    R1 最终 CAD 记录的 C6/C7/C8 到 MCU 最短 3V3 路径为 6.65/6.71/8.99 mm，C9 储能路径为 13.82 mm。它们是几何记录，不是“去耦已优秀”的证明；较长路径需要在纹波、瞬态和 EMI 实测中复核。

    1. 把每个电容分配给具体 U1 电源脚，而不是只写“靠近 MCU”；
    2. 优先缩短电源脚到电容再到地的完整环；
    3. 避免在电容与地之间插入长细线或多个无关过孔；
    4. 不要让快速信号穿过电容的高频回流区域；
    5. 对 C9 的位置同时考虑储能、电源主干和 C8 的机械间距。

    20.5 晶振器件应近、短、对称

    Y1、C11、C12 与 U1 的 OSC_IN/OSC_OUT 构成小信号振荡网络。它应远离 U3/L1 的开关区域和快速 USB 线，且不允许形成测试支路。

    • Y1 靠近 U1 的振荡引脚，两个信号路径短且几何相近；
    • C11/C12 对称放置，地端就近返回安静地；
    • 避免其他信号或分割地从晶振回路中间穿过；
    • 在周边保留地参考，但不要未经验证添加改变负载的任意铜结构；
    • 最终布线后重新检查 12 MHz 值与 1 脚/封装方向。

    最终组织坐标中 Y1 位于 (850, 140) mil，C11/C12 位于 (660, 140)/(1040, 140) mil。实际起振与负载匹配仍需上电测量。

    20.6 把负载开关放在输出连接器附近

    U4/U5 应位于其电源来源和 J2 输出之间，使 IN/OUT 主电流路径短而宽，同时让输出电容靠近负载开关和连接器。

    • U4 的 VBUS_IN → U4 → 5V_SW → J2 路径避免绕过整板；
    • U5 的 3V3_SYS → U5 → 3V3_SW → J2 路径考虑 U3 总负载；
    • ISET 与 EN 是小信号，远离高 di/dt 电源铜；
    • C4/C5/C13/C14 按各自输入/输出角色靠近相关引脚；
    • 保留 U4/U5 和电源走线散热空间，但不要假设铜面积已经完成热验证。

    20.7 把 GPIO 电阻组织成可读通道

    R14–R23 是重复接口通道。按 J2 引脚顺序排列，可以减少交叉、简化审图和测量，也让错误通道更容易定位。

    1. 让 R14–R21 按 GPIO0–GPIO7 顺序排列；
    2. R22/R23 对应 UART_TX/UART_RX，方向与 J2 标签一致；
    3. 保持相同器件方向和统一中心间距；
    4. MCU 侧和连接器侧网络名在布局标注或高亮中保持可追踪；
    5. 为探针接触焊盘留出空间，不让丝印遮挡。

    整齐排列不是美化目的本身；它降低布线分叉和人工检查复杂度，但不能以牺牲走线回流、板边或装配间距为代价。

    20.8 用受控间距对齐重复无源器件

    R1 最终把 C6–C9 对齐在 x = 680 mil 的竖直列，Y 坐标依次为 260、350、440、530 mil。R1/R2 分别位于 (399, 400)/(399, 544) mil。统一网格让机械间距和布线通道更容易复核。

    对齐时应先确定：

    • 本体最大尺寸与庭院宽度；
    • 相邻焊盘之间所需铜和阻焊间距；
    • 走线从哪一侧进入、地回流从哪一侧离开；
    • 元件值和位号在装配图中是否易辨；
    • 返修工具能否接触每个元件两端。

    若统一间距导致 C8 离 U1 电源脚更远，应记录并在性能测试中复核，而不是把视觉整齐写成电气最优。

    20.9 在紧凑、可布线、热与检查之间权衡

    布局权衡不能只优化单一指标
    目标过度追求的风险平衡方法
    面积最小本体重叠、无法探测或返修、回流被切断设置机械与操作裕量后再压缩
    路径最短高噪声和敏感网络被迫相邻按回路性质分区，而不是只比较欧氏距离
    铜最宽挤占地参考、扩大 BUCK_SW 天线面积按电流、压降、热和噪声分别设计
    排列最整齐关键电容离引脚变远关键回路优先，重复通道次之
    过孔最少走线绕远、切割地或造成拥堵比较完整回路和层转换成本

    布局评审应让电气、机械、制造和测试角色共同参与。单一视角优化通常把问题推迟到更昂贵的阶段。

    20.10 完成机械审核后才锁定布局

    锁定位置表示“任何移动都需要变更流程”，不是“以后绝不修改”。在锁定前必须完成：

    • 全部 1,128 元件对的本体与庭院检查；
    • J1/J2/J3 方向、外伸、插拔和外部配合审核；
    • C8/C9、R1/J1 等历史问题的专门回归检查；
    • USB、Buck、晶振、去耦和受控电源回路评审；
    • 板边、孔、探针、丝印与装配访问检查；
    • 删除或标记所有与旧位置相连的铜，再进入布线。

    移动器件后，旧走线不会自动变成正确的新走线。即使铜仍在同一网络上，它也可能悬空、穿过新器件、产生错误支路或连接旧焊盘位置。必须按受影响网络执行 rip-up、重新布线和完整 DRC。

    20.11 案例：修正 J1 方向与 C8/C9 重叠

    R1 组织化重布局的“现象—原因—检测—修复—预防”
    阶段J1 / R1C8 / C9旧布线
    现象J1 朝下，R1 与壳体冲突电容脚位/实体仍重叠元件移动后出现大量间距、差分对和连接错误
    根因只看中心/焊盘，没有审核真实开口和壳体只凭视觉移动，没有统一本体间距与列布局保留了针对旧坐标生成的铜
    检测3D、数据手册包络、插头方向本体/庭院全对检查DRC、独立连通分析和旧坐标铜审计
    修复J1 旋转 −90°朝左，R1/R2 移到安全列C6–C9 在 x=680 mil 按统一 Y 间距排列清除受影响铜，按新坐标完整重布并重开文档复核
    预防连接器优先、导入外部配合件先定义包络和接收间距布局变更单必须列出受影响网络和重布范围

    最终机械审核得到 48 个元件、0 本体重叠、0 庭院/形状重叠、最小接收裕量 10.748 mil；J1 只保留 −5.055 mil 的有意左侧外伸。随后旧铜被完整重布，最终路由为 608 条走线、130 个过孔，DRC 各类错误为 0。

    设计练习：选择一个需要移动的元件，写出其本体约束、受影响网络、需要删除的旧铜、回归检查和制造输出失效范围。未列出受影响网络前，不允许在发布板上移动。

    本章小结

    功能布局从固定连接器开始，依次保护入口、缩小关键回路、组织重复通道，并用机械全对审核收尾。元件位置一旦改变，旧铜和旧审核同时失效；只有重新布线并重跑电气、机械和制造检查，才形成新的可信基线。

    事实依据：最终 CPL、元件事实图谱、制造清单与检查清单。路径长度和间隙为 CAD 计算值；热、纹波、EMI 和实物插接仍待验证。
