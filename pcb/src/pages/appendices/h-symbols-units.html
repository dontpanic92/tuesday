H.1 符号、器件、footprint 与实体不是同一件事

原理图符号（symbol）
表达功能、针脚和逻辑连接；图上大小与真实器件无比例关系。
器件实例（component/device）
带唯一位号、值、料号和属性的设计对象，例如 R5、U1。
footprint / land pattern
PCB 上的焊盘、孔、阻焊、锡膏、丝印和装配边界定义。
package / 外壳
供应商器件的真实机械外形及引脚形式；相似外壳不保证针脚顺序相同。
3D body / 实体包络
用于插接、碰撞和高度检查；不能替代焊盘号与数据手册尺寸核对。

典型错误：符号针脚正确、footprint 焊盘号错误，ERC 仍可能通过；焊盘电气正确、实体包络重叠，DRC 仍可能通过。因此电气映射和机械审核必须分别完成。

H.5 PCB 图元与制造层

表 H-4　常见 PCB 对象
对象作用常见误解
Pad器件引脚焊接或电气接触的铜区。焊盘正确不代表实体不会碰撞。
Track / Trace线状铜导体。网络连通不代表线宽、回流或温升合格。
Via连接铜层的金属化孔。过孔有电阻/电感和占板空间，不是“免费换层”。
PTH金属化通孔，可焊接并导电。不能与 NPTH 混用。
NPTH非金属化机械孔。孔壁没有设计导电功能。
Board outline最终铣板边界。必须闭合；可见矩形不保证只有一个有效轮廓。
Solder mask覆盖非焊接铜面的阻焊层。不是铜层，也不是锡膏层。
Paste mask钢网锡膏开口。通孔元件通常不由该层定义焊料。
Silkscreen文字、方向和用户标识。不能依赖丝印作为唯一装配定位资料。
Native pour按网络和规则可重算的大面积铜。边界对象本身不是生成铜；修改后必须重铺并查 Gerber。
Static fill固定铜图元。不会随规则自动重算；R1 的 17 个 GND 区域属于此类。
Courtyard装配和返修预留边界。不是电气间距，也不一定等同器件实体。

H.9 书写与读数规范

  • 数值与单位之间留空格：3.3 V、100 nF、50 mm。
  • 单位符号不加复数、不随中文变化：写 10 Ω，不写 “10 Ωs”。
  • 网络名、位号和器件型号使用等宽样式并保持大小写：3V3_SYS、R12、SY8089AAAC。
  • 电压必须说明参考点，例如“3V3_SYS 对 GND”，不能只写“测得 3.3”。
  • 频率、占空比、纹波和温度应说明仪器、带宽、负载和工况。
  • 标称、典型、最小、最大、计算和实测不得混用。
