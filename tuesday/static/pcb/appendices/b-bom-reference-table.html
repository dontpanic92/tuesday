    B.1 如何使用本表

    本附录覆盖最终 PCB 上全部 48 个物理位号：R1–R23、C1–C14、U1–U5、J1–J3、L1、D1、Y1。最终采购 BOM 有 21 个分组行、46 个拟装配位号；CPL 和 PCB 有 48 行/位号，因为 J2、J3 是有焊盘但不随标准装配采购的 DNP 器件。

    最终 BOM 文件的 Value 列为空；下表的阻值、容值、频率和接口规格来自最终 EasyEDA 项目归档的器件元数据，而不是把空列猜成某个值。料号、封装和装配状态已与项目归档、BOM、CPL 交叉核对。

    B.5 采购、装配与检验时的使用规则

    1. 按位号核对，而不是只按数量：同一料号分组采购，但失效分析和返修必须回到单个位号。
    2. 值、料号、封装三者都要匹配：“0603”相同不代表电气值相同；SOT-23-5 的引脚排列和方向也不能仅凭外形判断。
    3. DNP 不是删除：J2/J3 的焊盘、孔、方向、机械空间和网络仍属于设计，不能因 BOM 没有它们而从审核中忽略。
    4. 方向敏感器件单独检查：U1 pin 1、U2/U3/U4/U5 的封装方向、J1 插接方向、D1 极性、Y1 脚位和 J2/J3 pin 1 都必须与装配资料一致。
    5. 库存分类会变化：供应商代码和基础/扩展料分类只代表发布时记录；下单前应重新确认可用性、生命周期和替代料的完整参数。

    覆盖检查：本页共有 23 个 R、14 个 C、5 个 U、3 个 J，以及 L1、D1、Y1，共 48 个唯一位号。若制造或返修清单的集合不同，应先解释差异，再继续。
