    学习目标

    铺铜不是“画一个大框然后变绿”。完成本章后，你应能区分原生铺铜边界、重铺后生成的铜和静态 FILL，正确设置网络、间距、热焊盘、优先级与孤岛策略，并通过源对象、DRC 和 Gerber 三层证据确认制造铜真实存在。

    • 解释铺铜边界与生成铜之间的关系；
    • 比较可重算的原生 EasyEDA 铺铜和静态填充；
    • 在每次几何或规则变更后重铺并审核；
    • 发现越出板框、孤岛、狭颈和“只有边界没有铜”；
    • 在独立 Gerber 查看器中确认实际铜区域。

    R1 的冻结发布源最终包含 0 个原生 POUR、17 个静态 GND FILL。本章推荐未来修改优先使用经过验证的原生铺铜，但不要在没有重新完成全部审核的情况下直接改写已发布 R1。

    24.1 铺铜边界做什么

    原生铺铜边界定义“工具可以在哪个区域尝试生成某个网络的铜”。它还携带层、网络、优先级、填充方式、最小颈宽和是否保留孤岛等属性。边界本身不是最终铜。

    EasyEDA 源格式把两者分开：

    • POUR：覆铜边框，保存网络、层、路径、顺序、类型和孤岛策略；
    • POURED：根据某个 POUR 和当前障碍/规则计算出的覆铜结果；
    • FILL：固定多边形铜，几何不会因为附近走线移动而自动重算。

    因此，看到一个可选中的边界只证明“存在一个生成请求”，不能证明工具已经得到有效 POURED 结果，更不能证明导出的 Gerber 中有铜。

    24.2 原生 EasyEDA 铺铜与静态填充图元

    原生铺铜与静态 FILL 的维护差异
    属性原生 POUR/POURED静态 FILL
    几何变更后执行重铺可按新障碍和规则重新计算保持旧多边形，必须人工再生成/修改
    间距由当前铺铜规则重新应用只有创建时或人工审核时的几何
    热焊盘/孤岛可由铺铜设置动态生成需要静态几何明确表达
    维护风险边界、重铺状态或生成失败可能被忽略元件/走线移动后容易遗留陈旧铜
    推荐用途持续编辑的 PCB 优先仅在有明确原因和完整再审核流程时使用

    R1 的最终制造铜由 17 个已连接、间距安全的静态 GND 区域组成：顶层 11 个、底层 6 个。它们已在 Gerber 中确认，但任何后续 PCB 几何修改都会使这份确认失效。

    24.3 间距、热焊盘、优先级、孤岛与网络分配

    创建铺铜时至少明确以下参数：

    网络
    R1 地铜应分配给 GND。无网络或错误网络铜可能成为浮铜或短路源。
    层
    顶、底层分别建立边界；不能假设一个对象自动覆盖两层。
    间距
    对不同网络焊盘、走线、过孔和板边应用制造规则。
    热焊盘（Thermal Relief）
    用辐条连接大铜与焊盘，改善可焊性；辐条宽度和数量仍需满足电流。
    优先级
    多个铺铜重叠时决定计算顺序；规则系统也可针对特定铺铜选择规则。
    孤岛
    与目标网络没有有效连接的铜块。除非有明确屏蔽目的和连接策略，通常删除。
    最小颈宽
    防止生成细长、难制造或高阻抗的铜丝。

    设置值必须与板厂能力、电流、可焊性和回流目标共同审核，不能仅采用软件默认值。

    24.4 优先使用可随几何重算的原生铺铜

    对仍会继续修改的设计，原生铺铜更容易保持规则一致：元件、过孔或走线移动后，重铺会重新避让障碍并更新热焊盘。静态填充则可能仍占据旧位置。

    1. 在板框完成并锁定后，于目标铜层创建边界；
    2. 明确选择 GND、层、间距、热焊盘、孤岛和优先级；
    3. 让边界位于闭合板框内，并给板边规则留出空间；
    4. 执行重铺，确认生成结果而不是只看边界；
    5. 运行 DRC并检查地连通、狭颈和敏感回流；
    6. 导出 Gerber，在独立查看器中确认实际铜。

    “优先原生铺铜”不是“原生铺铜永不出错”。R1 的真实故障恰恰说明边界坐标、重铺结果和导出铜都必须检查。

    24.5 把每个边界与闭合板框比较

    铺铜边界应采用与 PCB 相同的坐标、原点和单位，并在预期板框内。审核不能只缩放到“看起来大概覆盖板子”，而要计算边界包围盒并与单一闭合板框比较。

    1. 确认板框只有一个闭合片段且尺寸正确；
    2. 列出每个铺铜对象的 ID、层、网络和边界包围盒；
    3. 检查边界最小/最大 X、Y 是否落在板框或允许内缩范围；
    4. 检查多边形首尾闭合、方向、内洞和自相交；
    5. 检查边界是否因原点移动、单位混淆或复制粘贴出现在远处；
    6. 删除无法解释的重复、板外或无网络边界。

    隐藏或不可见的板外铺铜仍属于设计对象，可能影响导出、性能或后续编辑。不要仅因当前视窗看不到它就认为不存在。

    24.6 在任何几何或规则变化后重铺

    以下任一变化都会使旧铺铜结果失效：

    • 移动、旋转、替换或删除元件/封装；
    • 新增、删除或移动走线、过孔、焊盘和禁止区；
    • 改变线宽、间距、热焊盘、优先级、孤岛或网络类别；
    • 改变板框、原点、层设置或铜到板边规则；
    • 改变铺铜网络、边界或层。

    重铺后的流程是固定的：保存 → 重铺全部相关层 → 检查生成状态 → DRC → 地连通/回流人工审核 → 保存、关闭并重开 → 再次 DRC → 导出 Gerber。不要在重铺前保存一张截图，随后用它证明修改后的铜仍然正确。

    24.7 确认实际铜，而不是相信可见边界

    最少使用三层证据：

    铺铜证据链
    证据要确认的内容不能单独证明
    源对象边界网络、层、坐标和设置正确；生成结果对象存在导出的制造文件一定包含铜
    DRC/连通分析铜与目标网络连接，间距和孤岛符合配置Gerber 版本最新、板厂实际看到该铜
    Gerber 铜层最终输出中存在所需铜几何、热焊盘和避让源网络语义、原理图逻辑或实物焊接正确

    在编辑器里关闭边界显示，只显示生成铜，可以快速发现“只有轮廓没有填充”；再高亮 GND，检查每个区域是否真正连接到地焊盘或地过孔。

    24.8 在导出的 Gerber 中检查铜区域

    1. 重新生成 Gerber，不复用旧 ZIP；
    2. 在独立查看器打开顶层 GTL 与底层 GBL；
    3. 叠加板框、钻孔和阻焊，确认铜没有越出外形或侵入孔；
    4. 检查 GND 铜在 USB、MCU、Buck 和连接器区域的连续性；
    5. 放大热焊盘、狭颈、孤岛和板边避让；
    6. 统计或解析铜区域，并与源设计记录比较；
    7. 保存查看器截图、文件哈希和审核日期。

    R1 最终 Gerber 包含顶层 11 个、底层 6 个 G36/G37 实心 GND 区域，面积分别为 562.422 mm² 和 520.456 mm²。这证明发布 Gerber 中有这些铜几何；它不证明实物板已制造或地阻抗已经测量。

    24.9 案例：板外畸形边界与缺失的生成铜

    R1 铺铜故障的结构化复盘
    阶段事实
    现象一编辑器中存在原生铺铜边界，但没有生成可用的填充铜；边界的视觉存在容易被误当作地平面。
    现象二制造审核发现两个原生铺铜对象完全位于板外，ID 为 5296498eb95b5e66 与 d420c7c130568fae，分别涉及两层。
    根因类别边界坐标/包围盒畸形，加上没有把“重铺结果”和“制造铜”作为独立质量门。
    检测枚举所有铺铜对象并把包围盒与闭合板框比较；检查生成铜对象；在 Gerber 中查看实际铜区域。
    R1 修复删除两个板外原生边界；冻结发布保留 17 个已验证静态 GND FILL；重跑 DRC并重新生成制造包。
    未来预防优先创建正确的板内原生 GND 铺铜，任何修改后重铺；同时检查边界、生成结果和 Gerber，不再把轮廓当作铜。

    最终源与工程归档均为 0 个原生 POUR，Gerber 中仍有 17 个有效 GND 铜区域。这两句话并不矛盾：制造铜来自静态 FILL，而不是原生铺铜对象。

    24.10 地连续性、返回路径与孤立铜岛

    面积很大的地铜如果被信号通道切成狭颈，或完全不连接到 GND，也不能提供预期返回。审核时应沿关键路径观察地电流如何流动：

    • J1/U2 到 U1 的 USB 参考铜是否连续；
    • 信号换层处是否有附近地层转换；
    • U1 去耦地端是否能短路回到相应地脚；
    • U3 输入/输出脉冲地是否与敏感反馈路径共享狭窄阻抗；
    • 板边和过孔阵列是否形成细颈；
    • 每个铜岛是否通过焊盘、走线或过孔与 GND 主体连接。

    孤岛删除规则要配合人工复核。一个被工具保留的小岛可能只通过很细的颈连接；一个被删除的区域又可能暴露布局过于拥挤。结果本身应成为布局改进反馈。

    24.11 实验：移动走线、重铺并检查间距变化

    目标：在教学副本中观察原生铺铜怎样响应几何变化，并建立从编辑器到 Gerber 的证据链。不要修改冻结 R1 发布源。

    1. 在闭合板框内创建顶层 GND 原生铺铜，设置网络、间距、热焊盘和删除孤岛；
    2. 执行重铺，记录边界、生成铜、热焊盘和 DRC；
    3. 在 GND 铜中穿过一条其他网络走线，重铺并测量避让；
    4. 移动该走线但暂不重铺，观察旧铜为何已经不可信；
    5. 再次重铺，比较新旧铜边缘、狭颈和孤岛；
    6. 故意把副本中的边界移到板外，使用包围盒检查发现并恢复；
    7. 导出 Gerber，在独立查看器确认最终铜与板框；
    8. 保存源对象、DRC、Gerber截图和文件哈希。

    通过判据：学生能指出边界、生成铜和 Gerber 铜的区别，能证明移动走线后必须重铺，并能发现板外边界与孤立铜。

    本章小结

    铺铜是由边界、规则和当前几何计算出的制造铜，不是一个可见轮廓。持续编辑优先使用可重铺的原生铺铜；无论使用原生铺铜还是静态 FILL，都必须检查板框、网络、连通、DRC 和最终 Gerber。R1 的板外边界与缺铜经历说明，只有完整证据链才能放行。

    事实依据：R1 制造清单、检查清单、最终 Gerber/工程归档，以及 EasyEDA PCB 形状格式中的 POUR、POURED 与 FILL 定义。
