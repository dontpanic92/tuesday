    23. 电源、地与 Buck 转换器布线（Routing Power, Ground, and the Buck Converter）

    学习目标

    电源布线的目标不是“用最宽的线连起来”，而是在电流、压降、温升、瞬态回路、开关噪声和返回路径之间取得受控结果。完成本章后，你应能划分 VBUS、受控输出、Buck 关键回路和地缝合，并逐个审核去耦电容的物理回路。

    • 从电流、允许压降、温升和制造铜厚估算线宽；
    • 追踪 VBUS_IN、3V3_SYS、5V_SW、3V3_SW 的完整路径；
    • 区分 Buck 输入环、开关节点、输出环与反馈路径；
    • 使用地铜和缝合过孔缩短返回，而不制造孤岛；
    • 把热、纹波和瞬态明确标记为待实物验证。

    23.1 从电流与温升估算线宽

    线宽估算需要输入铜厚、外层/内层、环境、允许温升、长度、最大连续电流和可接受压降。在线计算器或经验表只在这些假设明确时有意义。

    1. 为网络写出正常、峰值和故障/限流电流；
    2. 取得计划生产的铜厚，不默认所有板都是 1 oz；
    3. 按允许温升估算最小截面，再用 V = I × R 检查压降；
    4. 把过孔、连接器、开关和焊点电阻纳入完整路径；
    5. 增加制造和温度裕量，避免长期工作在计算极限；
    6. 样板上电后以实际电流、压降和热像/温度测量修正假设。

    R1 采用 24 mil VBUS 主干、20 mil 3V3_SYS、16 mil 两条受控输出、12 mil GND 显式线和 14 mil BUCK_SW。这些是最终设计选择，不是对任何铜厚与环境都成立的额定电流声明。

    23.2 VBUS 与受控输出的电流路径

    把每条电源画成“源 → 保护/开关 → 负载 → GND 返回 → 源”的闭合路径：

    • VBUS_IN：J1 VBUS 进入，供 U3 Buck 与 U4；
    • 3V3_SYS：U3/L1/C3 输出，供 U1、逻辑电路与 U5；
    • 5V_SW：VBUS_IN 经 U4 到 J2-3，典型设计限流约 0.402 A；
    • 3V3_SW：3V3_SYS 经 U5 到 J2-4，典型设计限流约 0.252 A；
    • 所有负载电流经 J2 GND、地铜/地线返回 J1 和相关电源器件。

    两路限流目标不能绕过 USB 总输入预算。即使 U4 的限流约 0.4 A，MCU、Buck 损耗和 3V3_SW 也共享主机供电；真实主机、线缆、连接器和温升必须测量。

    5V_SW 是电源输出，不是 5 V 逻辑。不要用加宽电源走线来暗示 GPIO 可以承受或输出 5 V。

    23.3 Buck 输入环、开关节点、输出环与反馈路径

    四条路径的优化目标不同：

    U3 Buck 的关键物理路径
    路径主要元件布局/布线目标
    输入脉冲环C2、U3 VIN/开关、U3 GND极短、宽、低电感；C2 紧靠引脚
    开关节点U3 SW → BUCK_SW → L1短、无支路、铜面积受控，远离敏感网络
    输出环L1、C3、负载、GND 返回低阻抗、短回路，3V3_SYS 主干清楚
    反馈路径输出取样、R5/R6、U3 FB从安静输出点取样，远离 SW 与大电流铜

    BUCK_SW 使用 14 mil 并不意味着应铺成大面积铜。对高 dv/dt 节点，较大面积会增加对周围网络和空间的电容耦合。

    23.4 让开关噪声远离 USB、晶振与反馈

    U3/L1 周围的快速电压和电流变化可能通过电场、磁场、公共阻抗和地回路耦合到敏感网络。

    • 不要让 USB D+/D− 与 BUCK_SW 长距离平行；
    • 不要把 Y1、C11/C12 或 OSC_IN/OUT 放在电感和开关铜旁；
    • NRST、BOOT0 和 FB 走线避免经过开关节点边缘；
    • 输入/输出电容地端就近闭合，不把脉冲电流送入 MCU 地路径；
    • 地铜要连续，但不要让开关节点下方的寄生耦合未经评估扩散到大面积；
    • 布局完成后联合显示关键网络、地铜和器件本体进行区域审核。

    DRC 只能检查设定的几何规则，不会告诉你某段反馈线是否拾取了足以引起纹波或不稳定的噪声。

    23.5 地缝合与层转换

    地缝合过孔连接顶、底层地铜，为信号换层和局部高频回流提供较短路径。它们应服务明确的返回需求，而不是均匀撒满板面后假设问题自动解决。

    1. 在 USB 对换层、连接器入口和地铜狭颈附近检查是否需要地过孔；
    2. 在去耦电容和 IC 地脚附近提供低电感层间连接；
    3. 围绕板边或噪声区域的缝合密度应由频率、结构和 EMI 目标决定；
    4. 确认每个地过孔真正连接到两层已生成的 GND 铜；
    5. 避免地过孔切断其他关键布线通道或侵入庭院；
    6. 铺铜更新后重新检查孤立铜和过孔连通。

    R1 最终有 32 个地缝合/树状过孔，并记录 USB 过孔到最近地转换的最大距离为 155.33 mil，供样板验证复核。这个距离不是通用合格阈值。

    23.6 热铜与压降

    电源器件的热会通过封装引脚、焊盘、铜和空气散出。增加连接到散热引脚的铜通常能降低热阻，但不同网络和节点不能为了“散热”任意扩大。

    • U3、U4、U5 的 IN/OUT/GND 铜按数据手册建议和电流路径规划；
    • L1 的电流额定值、直流电阻和温升与铜布局共同审核；
    • 连接器、过孔和收颈处可能成为压降热点；
    • BUCK_SW 扩铜受 EMI 约束，不能照搬普通散热焊盘策略；
    • 热仿真或经验只能生成预期，原型仍需在额定负载下测温；
    • 同时记录环境温度、输入电压、负载、持续时间和测量方法。

    R1 已证明制造几何和规则通过，尚未证明 Buck 温升、负载开关压降、限流容差或满载热性能。不要把宽线和大铜面写成“热验证完成”。

    23.7 逐个检查去耦电容的物理回路

    为 C6、C7、C8 和 C9 分别填写回路审计，而不是把它们合并成“MCU 附近有四个电容”。

    去耦/储能回路审核表
    器件角色CAD 最短 3V3 路径还需检查
    C6100 nF 高频去耦6.65 mm对应电源脚、地回路、过孔电感
    C7100 nF 高频去耦6.71 mm对应电源脚、地回路、共享铜阻抗
    C8100 nF 高频去耦8.99 mm移动后回路、C9 间距与噪声表现
    C9较低频储能13.82 mm主干压降、负载瞬态与本体空间
    1. 从 U1 电源脚沿铜到电容电源端；
    2. 从电容地端沿地铜/过孔到 U1 地脚；
    3. 标出回路中的狭颈、过孔、共享路径和地缝；
    4. 确认没有被其他信号切断的返回区域；
    5. 把较差路径列为示波器测量与 R2 改进候选。

    23.8 练习：在 PCB 上标出高电流与高 di/dt 回路

    目标：在不修改冻结设计的 PCB 截图或副本上，把功能网络转换为物理电流环。

    1. 用一种颜色标出 J1 → U4 → J2 的 5V_SW 去程和 GND 返回；
    2. 用第二种颜色标出 U3 输入脉冲环；
    3. 圈出 BUCK_SW 铜和 L1；
    4. 标出 L1 → C3 → 负载 → GND 的输出环；
    5. 画出 R5/R6 到 FB 的取样路径，并标注应远离的噪声源；
    6. 任选 C6–C8，画出完整去耦回路；
    7. 指出至少一个地缝合过孔服务的返回路径。

    通过判据：每个环同时包含去程和返回，能指出高电流与高 di/dt 的差别，并且没有把“GND 符号”当作电流到此消失。

    本章小结

    电源与地布线必须按闭合回路设计。VBUS 和受控输出关注电流、压降与热，Buck 还要控制输入环、开关节点、输出环和反馈路径，去耦则关注芯片引脚附近的高频回路。CAD 可以验证几何，真实纹波、温升和瞬态仍需实验。

    事实依据：最终工程、制造清单、检查清单和项目电源事实。所有路径长度、线宽、铜和过孔数量为设计/CAD 事实；性能指标未实测。
