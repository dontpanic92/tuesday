学习目标与事实状态

本章沿最终 R1 数据路径 J1 → U2 → R3/R4 → U1 解释 USB 差分信号、ESD 导流、串联阻尼、回流路径和几何匹配。

状态约定：网络和值为【设计】；CAD 长度与偏斜为【计算】；信号完整性和失效后果为【预期】；受控阻抗、眼图、EMI 与 ESD 等级均为【待验证】。当前没有板级实测波形。

8.1 用直观语言理解差分传输

USB D+ 与 D− 构成差分对。接收器主要关心两根线之间的电压差，而不是只把任意一根线相对地的电压当作数据：

若 D+ 上升而 D− 下降，Vdiff 明显变化；若外界噪声几乎同样抬高两根线，变化更多体现在 Vcm。只有两条路径足够对称、接收器保持在线性工作范围且地参考连续时，共模抑制才真正有效。

8.2 共模噪声与差分数据为何不同

差分数据是两线之间有意制造的相反变化；共模噪声则倾向于让两线同向变化。实际 PCB 上，两者会互相转换：线宽或间距不一致、单边过孔、单边支路、不同参考平面、器件寄生不对称，都可能把共模能量变成差模误差，或把差模能量辐射成共模噪声。

差分路径中应保持对称的项目
项目不对称的后果【预期】
R3 与 R4 的阻值、封装和位置两线边沿与损耗不同，差分波形失衡。

8.3 U2：USB 数据线的双通道 ESD 导流器件

U2 是 USBLC6-2SC6，SOT-23-6 封装【设计】。它不是串联保险丝，也不会“吸收所有静电”；它为短时高压事件提供比 U1 输入更可控的导流路径，以 VBUS_IN 和 GND 为钳位参考。真实抗扰度取决于器件本身、焊接、入口布局和回流电感，R1 尚未做实物 ESD 试验【待验证】。

U2 缺失或开路时，USB 在安静环境中可能仍能通信，但入口 ESD 防护显著下降；U2 短路、漏电或方向错误时，D+/D− 或 VBUS 可能被钳住，导致无法枚举【预期】。

8.4 为什么 ESD 器件必须靠近连接器

静电脉冲边沿很快。连接器到 U2 之间的铜越长，寄生电感越大，电流越可能先在板内产生较高局部电压，并沿其他路径耦合到 MCU、晶振或电源。理想次序是：

  1. 脉冲从 J1 进入；
  2. 立即到达 U2；
  3. 通过短、宽、低电感的参考路径流向 GND/VBUS_IN；
  4. 受保护的数据线再继续前往 R3/R4 和 U1。

最终 CAD 记录的 J1→U2 段长度为 D− 700.884800 mil、D+ 700.884807 mil，即约 17.802474 mm；两线偏斜为 0.000007 mil【计算】。长度相等只说明几何对称，不能证明未保护段足够短或 ESD 合格。U2 位于连接器入口之后且布局等级为关键，但最终效果仍为【待验证】。

实验限制：不要用打火机压电点火器、裸线摩擦或人体放电“测试 ESD”。合规 ESD 试验需要规定的发生器、耦合网络、接地点、等级、重复次数和监督。

串联电阻与驱动器输出阻抗、走线阻抗和负载共同影响反射。它们不是“越大越安全”：阻值过大虽然可能减弱高频振铃，却也会降低边沿速度和眼图裕量。22 Ω 是最终 R1 设计值，不是实测调优结论。

8.6 线宽、间距、偏斜、支路与过孔

R1 的 USB D+/D− 使用 8 mil 走线【设计】。实际差分阻抗由铜厚、介质厚度和介电常数、线宽、两线间距、阻焊和附近铜共同决定；目前没有生产叠层，因此不能把“8 mil”直接等同于某个受控阻抗值。

  • 两线是否保持耦合和相同参考地；
  • 器件焊盘处是否形成长支路；
  • 换层时两线是否同步，并有邻近地过孔帮助回流换层；
  • 是否跨越地铜缺口、静态填充边界或其他回流中断。

8.7 地参考与回流连续性

差分对并不意味着“不需要地”。理想的两线电流会部分互相回流，但器件输入输出、电容、电缆屏蔽、共模分量和不平衡电流仍需低阻抗参考。若下面的地铜被切断，回流就会绕行，使环路面积、串扰和辐射增加。

R1 最终有 17 个连接到 GND 的静态 FILL 区域和 32 个 GND 缝合/树状过孔【计算】；这些铜已在 Gerber 中确认存在，但不是可自动重铺的原生 POUR。几何修改后必须重新生成并复核，否则“屏幕上看见边界”不等于制造铜仍连续。

8.8 为什么长度匹配不能单独保证信号完整性

R1 的三段几何偏斜极小【计算】，这是正确的几何审核结果，但还缺少以下证据：

  • 真实生产叠层和受控差分阻抗；
  • 焊盘、U2、R3/R4、过孔和连接器的寄生模型；
  • 真实线缆、主机和温度条件下的 USB 眼图；
  • 持续传输、挂起/恢复、重连、EMI 与 ESD 测试。
错误推理：“D+ 与 D− 等长，所以 USB 一定合格。”
正确表述：“CAD 几何偏斜已计算并很小；阻抗、损耗、共模和协议裕量仍待实物验证。”

8.9 实验：比较单端波形与差分波形

目的与方法

观察两条互补信号的单端波形，并用示波器数学通道得到 Vdiff。初学实验优先在隔离的低压教学信号源上完成；直接测量运行中的 USB 仅限具备合适差分探头和监督的实验室。

安全要求

  • 普通台式示波器探头地夹通常与保护地相连；不得把地夹接到 D+ 或 D−。
  • 不要用长鳄鱼夹地线探测 USB；它会显著增加环路和振铃，甚至造成短路。
  • 若测 R1，使用高阻、低电容差分探头或两根严格匹配的有源探头，并遵守探头共模范围。
  • 不要把本实验结果称作 USB 合规眼图；没有规定夹具、模板和校准就不能作合规判定。

步骤

  1. 让教学信号源输出幅度相同、相位相反的两路低压方波。
  2. 同时显示通道 A、通道 B，并设置数学通道 A−B。
  3. 给两路同时叠加一个小的同向扰动，比较单端波形与 A−B 的变化。
  4. 有条件时再在 R1 上记录 U2 前后或 R3/R4 两侧的波形；每次只改变一个测量点。

本章小结

  • USB 信息主要由 D+ 与 D− 的差分量承载，共模噪声能否被抑制取决于路径对称。
  • U2 把数据线 ESD 事件导向 VBUS/GND 参考；R3/R4 各为 22 Ω 串联阻尼。
  • R1 的三段 USB 几何偏斜已计算且很小，走线宽度为 8 mil。
  • 这些 CAD 结果不证明受控阻抗、眼图、EMI 或 ESD 合规。

事实来源

最终 EasyEDA 工程归档中的 J1、U2、R3、R4、U1 网络；R1 fabrication manifest 与 checklist 的 USB 几何和地回流记录；结构化事实库中的 usb_geometry、routing.widths_mil 与元件图谱。U2 机理只采用已复核的双线低电容 ESD 拓扑，不宣称未记录的实测等级。
