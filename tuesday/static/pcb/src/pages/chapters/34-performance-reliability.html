学习目标与证据等级

功能实验回答“能不能工作”，性能与可靠性测量回答“在什么条件下、以多大裕量、能否重复地工作”。本章要求同时记录激励、负载、环境、仪器设置和原始波形；没有这些上下文，一个“3.3 V”或“USB 正常”几乎没有工程价值。

R1 事实库的实测条目仍为 0。Buck 纹波/瞬态/温升、开关压降、PWM 抖动、UART 误码、USB 传输/挂起/线缆、EMI 和 ESD 全部待实物验证。本章只给方法、记录格式和判定逻辑，所有结果栏不得预填。

34.2 探头接地与避免测量引入的错误

测量系统也是电路的一部分。普通 10× 探头约有数 pF 电容，长地线形成电感环路；二者会制造或放大振铃。台式示波器 BNC 外壳通常接保护地，把地夹接到 BUCK_SW 等非地节点会直接短路该节点。

  1. 先确认被测点与仪器地的参考关系。所有单端探头地只接 R1 的 GND。
  2. 纹波测量优先在 C3、C4 或 C5 的正负焊端直接使用地弹簧，环路尽可能小。
  3. 记录探头倍率、耦合方式、带宽限制、采样率、记录长度和示波器输入阻抗。
  4. 同一波形至少保存“全带宽”和“20 MHz 带宽限制”两份；后者有利于比较电源规格，前者用于发现高频尖峰。
  5. 先用被动探头测一个低风险数字节点确认极性和尺度，再接关键电源节点。
  6. 直接测晶体脚会改变负载电容。Y1 起振/频偏优先用低电容有源探头或间接时钟输出验证。

不要用两根普通探头的地夹分别接在两个“浮动”节点上。高侧或浮动差分测量必须使用额定值合适的差分探头，或采用经审查的双通道相减方法且两个地夹都接 GND。

34.3 Buck 纹波与瞬态响应

目的

测量 U3/L1/C3 生成的 3V3_SYS 在不同负载下的稳态纹波、启动过冲、负载阶跃跌落/过冲和恢复时间。

原理

Buck 电感纹波近似受 VIN、VOUT、占空比、L1 = 2.2 µH 与开关频率影响；C3 = 22 µF 的有效电容、ESR/ESL 和高频回路决定输出纹波。公式只能预测趋势，不能替代板上测量。

器材与量程

  • ≥50 MHz 示波器，采样率 ≥500 MS/s；10× 地弹簧探头，另备低噪声同轴探测头更佳。
  • 5.0 V 限流电源、动态电子负载、DMM；负载上限由第 33 章验证包络决定。

安全与停止条件

不探测 BUCK_SW，除非使用额定差分探头和专门计划。出现 3V3_SYS 超出预定窗、持续振荡、U3/L1 快速升温、限流或 USB/MCU 复位时停止。

接线指导

CH1 探头尖端 → C3 的 3V3_SYS 端
CH1 地弹簧   → C3 的 GND 端
电子负载     → 经验证的 3V3 输出负载路径
CH2（可选）  → 负载触发或 3V3_SW，地接 GND

上电前检查

  1. 校准探头，设置 10×；确认地弹簧不碰邻近焊盘。
  2. 填写 3V3_SYS 允许纹波、过冲、跌落和恢复时间；若尚无要求，标为“表征”。
  3. 负载从 0 A 开始，记录环境温度和台式电源端/板端 VBUS。

编号步骤

  1. 在板空闲、USB 活动和经批准的 25%、50%、80% 负载点分别保存全带宽与 20 MHz 限带波形。
  2. 每个点测量平均值、峰峰值、主频和是否有低频包络；不要只读取自动测量一个数字。
  3. 从 0 V 捕获受控上电，记录 3V3_SYS 上升、过冲和建立时间。
  4. 做 10%↔50%、10%↔80% 负载阶跃，至少捕获 20 次并保存最差波形。
  5. 改变探头地线长度做一次对照；若尖峰随探法大幅变化，将其标为测量伪影候选。

预期结果（非实测）

预期 3V3_SYS 围绕约 3.3 V 稳定，负载阶跃出现有限跌落/过冲后恢复。R1 的真实纹波、主频、过冲和恢复时间均未知。

34.4 额定负载下的热测量

目的

在已经电气验证的负载包络内，测量 U3、L1、U4、U5、U1 和连接器的温升及达到热稳态的时间。

原理

器件温升由损耗、铜扩散、气流和环境温度共同决定。红外相机读数受表面发射率影响，反光金属会低报；热电偶会引入局部热沉。表面温度也不等于芯片结温，需要数据手册热模型。

器材与量程

  • 红外相机或细线热电偶，范围至少 20～100 °C；哑光高发射率胶带用于可安全贴附的测点。
  • 电子负载、限流电源、DMM、环境温度计；无强制风冷，除非测试计划要求。

安全与停止条件

“额定负载”必须来自已批准需求，不可直接把计算限流当连续额定电流。任何器件表面超过风险评估门限、温度 5 min 内仍持续快速上升、轨电压失控或材料变色时停止。

接线指导

使用与第 33.7 节相同的短粗负载线；热电偶线固定后不得跨接 J1、U3 或 L1 引脚。相机垂直角度、距离、发射率和环境反射条件保持一致。

上电前检查

  1. 填写每个器件的表面停止温度和由数据手册推导的结温评估方法。
  2. 在断电板上比较红外与接触温度，校正发射率。
  3. 确认该板已通过目标负载的短时电气测试。

编号步骤

  1. 记录环境温度，空载运行 10 min，每分钟记录六个测点。
  2. 施加 25%、50%、80% 连续负载；每档保持到 5 min 内温升小于预定稳定判据或达到最长时间。
  3. 对 5V_SW 与 3V3_SW 分开测，再按真实使用场景评估同时负载。
  4. 在最热工况保存热像和轨电压；卸载后记录冷却曲线。
  5. 完全冷却后复测功能，检查热应力是否留下漂移。

预期结果（非实测）

预期温度随负载上升并趋于稳态；最热点取决于哪一路消耗功率。R1 没有任何实际温度或连续额定负载结论。

34.5 PWM 时序准确度与抖动

目的

在长记录中量化 PWM 频率准确度、占空比分辨率、周期到周期抖动和多通道相对时序。

原理

平均频率偏差主要来自时钟源与定时器分频，短期抖动来自时钟噪声、中断或更新方式。12 MHz Y1 是设计值，其负载匹配与实际频率未验证；CAD 无法证明时序准确度。

器材与量程

  • 频率计或带统计功能的示波器，参考时基精度优于待验证要求至少 4 倍。
  • 四通道逻辑分析仪，采样率至少最高 PWM 的 50 倍；短线、共同地。

安全与停止条件

无负载测量，输入限于 0～3.6 V。若探头引入复位、USB 断开或异常过冲，停止并改善探测。

接线指导

J2 pins 5～8 分别接四通道，所有地接 J2 GND。若测晶体贡献，不直接给 Y1 加普通探头，而用 PWM/时钟输出的长时平均反推。

上电前检查

  1. 记录仪器时基校准日期和不确定度。
  2. 记录固件定时器时钟、计数值、更新机制和预期量化误差。
  3. 预先定义频率误差、占空比误差和峰峰抖动限值。

编号步骤

  1. 选择低、中、高三个频率，各测 10%、50%、90% 占空比。
  2. 每个点记录至少 10,000 周期的平均、最小、最大和标准差。
  3. 同时开启四路相同占空比，测频率一致性与边沿相对偏移。
  4. 在 USB 空闲、持续传输和负载阶跃三种系统活动下重复，观察干扰。
  5. 执行频率/占空比在线更新，捕获是否出现一次性异常周期。

预期结果（非实测）

四路应共享频率，平均误差受时钟与计数器量化限制；系统活动不应产生丢周期或危险毛刺。实际数值待测。

34.6 UART 误码率测试

目的

在不同波特率、线长和系统负载下统计 UART 数据错误，建立适用条件，而不把一次短回环的零错误推广为“永不误码”。

原理

观测误码率可写为 BER = 错误位数/总传输位数；若只按字节校验，应明确报告“错字节率”。观察到零错误并不表示真实 BER 为零，只给出与样本量相关的上界。

器材与量程

  • 3.3 V USB-UART 或第二块已校准设备、伪随机序列发生/校验软件。
  • 0.1 m、1 m 及目标应用线缆；逻辑分析仪采样率 ≥16×baud。

安全与停止条件

共同地先接，不接适配器 5 V；不使用 RS-232 电平。出现地电位异常、输入超出 0～3.6 V、板复位或线缆发热时停止。

接线指导

板 TX→对端 RX、板 RX←对端 TX、GND↔GND；使用外部回环或双向测试，但避免两个 TX 相连。长线测试保持远离开关电源磁性器件并记录布线方式。

上电前检查

  1. 选择每个工况的总字节数、校验算法和超时；记录种子以便重放。
  2. 在短线最低波特率下验证测试工具本身。
  3. 定义允许 BER/错字节率；没有要求则只表征。

编号步骤

  1. 短线下在低、中、高三个支持波特率各发送预定数据量。
  2. 分别在 USB 空闲、USB 持续传输、PWM 全开和受控电源负载阶跃时重复。
  3. 更换 1 m 和目标线缆，保持波特率不变；记录错误与波形。
  4. 对每个错误保存前后至少一帧和 UART 状态计数器。
  5. 恢复短线基线，确认测试后没有永久变化。

预期结果（非实测）

在批准线长和波特率内应满足预先定义的错误率；更长线或更高波特率可能降低裕量。R1 没有 UART BER 实测数据。

34.7 USB 传输、线缆、挂起/恢复与重连测试

目的

建立跨主机、端口、线缆和电源状态的 USB 回归矩阵，测量传输错误、持续速率、挂起/恢复和反复重连可靠性。

原理

USB Full-Speed 链路的可用性由物理层、主机调度、端点大小、固件缓冲和操作系统驱动共同决定。应用吞吐量不是 12 Mbit/s 线速；也不能用单一线缆结果证明兼容性。

器材与量程

  • 至少两类主机/操作系统、直接端口和已知良好集线器。
  • 三条已编号数据线：短标准线、较长合格线、已知良好对照线。
  • 带校验的双向传输工具、主机 USB 日志；可选协议分析仪和电流记录仪。

安全与停止条件

每次只有一个 VBUS 源。主机报告过流、J1/U2/U3 发热、端口保护动作或板持续复位时停止并回到限流电源。

接线指导

主机→被测线缆/集线器→J1；J2 只接经验证的小负载，J3 不供电。保持线缆不受拉扯，记录 USB-C 插头方向。

上电前检查

  1. 定义每个矩阵单元的枚举时限、传输字节数、允许错误、重连次数和挂起时间。
  2. 清空并开始记录主机日志，核对固件版本。
  3. 先用对照线和直接端口验证测试软件。

编号步骤

  1. 对每个主机×线缆×拓扑组合执行冷插拔并记录枚举。
  2. 进行固定时长双向传输，记录有效字节、耗时、校验错误和超时。
  3. 活动传输中发起 suspend，保持计划时长后 resume，验证安全状态与数据恢复。
  4. 每个组合执行 50 次软件关闭/打开端口和 20 次物理重连，统计失败。
  5. 翻转 USB-C 插头方向重复关键组合，检查可翻转路径。
  6. 保存失败单元的系统日志、供电电流和第一个异常时间。

预期结果（非实测）

预期所有批准组合稳定枚举、正确传输并从挂起/重连恢复；实际吞吐、错误和恢复时间未知。R1 的 CAD 长度不证明这些结果。

分析

按主机、线缆、方向和集线器分层。仅某条线失败可能是线缆；仅某插头方向失败优先查 J1 两组触点/CC；所有组合在高负载失败则联查电源完整性。

故障排查

保留主机第一个错误，不反复插拔覆盖日志。先确认 VBUS/复位/时钟，再查 U2、R3/R4、焊点和固件。协议分析仪显示重试不等于一定是 PCB，也可能是固件未及时服务端点。

34.8 ESD 与 EMI 预兼容概念

预兼容（pre-compliance）是在正式认证前寻找明显风险，不等于认证。R1 的 U2 USBLC6-2SC6 保护 USB D+/D−，但外部 GPIO 没有专用 ESD 阵列；两层板使用 17 个已在 Gerber 中确认的静态 GND FILL，而非可重铺原生铜区。USB 受控阻抗、眼图、EMI 与 ESD 均未验证。

34.9 生产容差与多板测试

一片样板只能说明“这片在这些条件下的结果”。电阻、电容、晶体、开关阈值、焊接和板材会产生散布。初步工程样本可选择至少 5 片观察趋势，但它不能替代为量产良率设计的统计样本；样本量应由风险、预期变异和置信目标决定。

  1. 从不同拼板位置/包装位置随机取样，不只挑外观最好的一片。
  2. 所有板使用同一治具、固件、仪器设置、环境和步骤；随机化测试顺序，减少温度漂移偏差。
  3. 每片记录个体结果，不能只保留平均值。规格判定通常要求每片都通过。
  4. 计算平均、最小、最大、极差和标准差；样本少时明确说明统计不稳定。
  5. 对离群点先复测治具，再隔离板做失效分析；不得直接删除数据。

本章小结

可靠测量要求正确仪器、低扰动探测、预先声明的判据和完整原始数据。R1 的性能数字仍全部待测；只有把单板波形扩展为工况矩阵、多板散布和可验证的 R2 要求，测量才真正进入工程闭环。
