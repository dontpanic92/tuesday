学习目标

USB D+ 与 D− 是一对共同工作的传输路径。完成本章后，你应能从真实生产叠层确定差分几何，按 J1—U2—R3/R4—U1 的顺序布线和测量三段路径，并准确说明“长度匹配”“阻抗”“眼图”和“EMI”是不同层次的证据。

  • 根据板厂叠层建立线宽、间距和参考层假设；
  • 保持 D+、D− 耦合、对称、同参考和同层转换；
  • 缩短连接器到 ESD 的未保护段并避免支路；
  • 测量三段长度与偏斜，而不迷信显示的小数位；
  • 把眼图、EMI 和 ESD 明确留给实物验证。

22.1 从真实叠层确定线宽与间距

差分阻抗由铜厚、介质厚度、介电常数、线宽、线间距、到参考铜距离、阻焊和制造公差共同决定。只知道“两层板”和“8 mil 线宽”不足以计算并保证生产阻抗。

  1. 向计划使用的板厂取得具体板厚、铜厚、介质参数和阻抗服务条件；
  2. 说明 D+/D− 位于哪一层、参考哪一片连续 GND 铜；
  3. 使用板厂计算器或场求解器得到候选线宽/间距；
  4. 检查候选几何能否从 J1、U2、R3/R4 和 U1 焊盘实际逃逸；
  5. 把差分规则、允许局部例外和制造公差写入发布资料；
  6. 叠层变化后重新计算，不沿用旧板的 8 mil 数字。

R1 的最终 D+/D− 使用 8 mil 走线，但发布资料没有受控阻抗叠层。因此本章只把它作为设计几何事实，不把它写成“90 Ω 已验证”。

22.2 保持耦合、对称与相同参考

差分对传输的是两线电压差，但每条线仍与周围参考和结构发生电磁耦合。两线应尽量经历相同环境：相同层、相似拐角、相似邻近铜、相同数量和位置关系的过孔，以及连续 GND 参考。

  • 成对布线，不让一条线先穿过拥堵区、另一条走完全不同路径；
  • 保持规则间距，短暂扇入焊盘后尽快恢复；
  • 避开板边、地缝、连接器固定孔和开关节点；
  • 若必须换层，两线在相近位置成对换层，并提供附近地过孔；
  • 避免一条线靠近铜岛、另一条线靠近空洞造成参考不对称；
  • 不要为了视觉完全平行而穿过更差的参考区域。

22.3 缩短连接器到 ESD 的未保护段

J1 到 U2 的铜在 ESD 保护器件之前。该段越长，静电事件越可能耦合进板内其他网络，也越容易形成支路和更大的放电回路。

  1. 先固定 J1 朝向和外壳脚，再把 U2 放在数据引脚进入板内的位置；
  2. D+、D− 从 J1 直接进入 U2，不先绕过 MCU 或测试点；
  3. U2 的 GND 端建立短、宽、低电感回路，并就近连接地铜/地过孔；
  4. 不要把未保护段与 BUCK_SW、晶振或长外部 GPIO 平行走线；
  5. 若需要测量点，评估其焊盘和支路对信号与 ESD 路径的影响。

“ESD 器件离连接器近”应由实际铜路径和回流证明，不能只比较两个封装中心。

22.4 串联电阻相对 MCU 的位置

R3/R4 是 22 Ω 串联电阻，最终布局要求它们成对并靠近 MCU 端。这样可以让电阻后的 USB_DP_MCU/USB_DM_MCU 段保持短，并把阻尼元件放在驱动端附近。

  • R3/R4 使用相同方向、相似焊盘逃逸和等效拓扑；
  • 不要在电阻两侧形成同名网络旁路；
  • 电阻前后分段测量，避免把器件本体长度误算成铜线；
  • 保持电阻到 U1 PA11/PA12 的路径无支路、同参考；
  • 最终是否需要 22 Ω 以及最佳值，仍应结合 MCU 指南和实物信号测试。

22.7 长度匹配与偏斜解释

几何偏斜定义为 |LD+ − LD−|。长度更接近通常有助于降低两线到达时间差，但传播速度还取决于局部介质和参考结构；两条几何等长、环境不同的线仍可能有电气不对称。

R1 记录的偏斜小于 0.00006 mil，说明 CAD 生成路径在数学上高度匹配。这个数字远小于常规 PCB 制造和材料公差，不能按其小数位宣称物理板也达到同样差值。工程结论应是“几何匹配没有成为主要差异源”，而不是“传播时间完全相同”。

若 D+ 为 500 mil、D− 为 497 mil，则几何偏斜为 3 mil。修正前先找出差异来自焊盘扇出、过孔还是主路径；不要在任意空地添加密集蛇形线而忽略参考和自耦合。

22.8 极小几何偏斜不等于阻抗保证

长度匹配只回答“几何路径差多少”。阻抗还取决于叠层与局部环境，插入损耗取决于材料、铜粗糙度和长度，反射取决于连接器、焊盘、过孔和宽度变化，共模还受不对称与返回路径影响。

DRC 的“差分对 0 错误”只说明已配置规则通过，不自动升级为后三种结论。

22.9 把 USB 眼图与 EMI 留给实物验证

制造前应保存叠层假设、线宽/间距、长度、过孔、参考铜和 DRC 结果。制造后再验证：

  • 不同主机、线缆和端口上的枚举、重连、挂起/恢复；
  • 持续数据传输和错误/掉线情况；
  • 按 USB 测试方法进行眼图或相关电气测量；
  • 近场探测与辐射/传导 EMI 预扫描；
  • USB 端口及外壳的 ESD 鲁棒性；
  • 板批次、温度和电源条件变化。

探测 USB 差分对会增加电容和支路，错误探头或长地线本身可能破坏波形。测试方法、夹具和带宽应由具备高速测量经验的人员制定。

本章小结

USB 差分布线需要真实叠层、耦合对称、连续参考、正确器件顺序和分段测量。R1 的极小 CAD 偏斜是几何事实，不是阻抗或合规证书；最终信号质量必须由生产条件和实物测试补齐。

事实依据：R1 制造清单、检查清单、最终工程和 CAD 偏斜公式。所有长度均为计算值；阻抗、眼图、EMI 与 ESD 尚未实测。
